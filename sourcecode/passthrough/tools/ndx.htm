Tools for Loading your Driver

This provides you, the user, some way to get the system to load your driver. You must (a) create a service entry for your driver and (b) specify that service name as an upper filter of the mouclass service.

The files:
  1. RegisterDriverService.cpp
  2. MouFiltrServiceDescription.reg
  3. rds.exe
  4. Use RegEdit to edit the registry, and Visual Studio to compile the executable.

What does it do

Both the .reg file and executable add a new service entry called "moufiltr." Even if you use one of these files, you must still open HKLM\System\CurrentControlSet\Control\Class\{4D36E96F-E325-11CE-BFC1-08002BE10318} and modify the value of "UpperFilters" to include "moufiltr" BEFORE mouclass. It should read "moufiltr\0mouclass\0\0" -- treat \0 as a newline in the registry editor.
